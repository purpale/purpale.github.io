# 走近科学之消失的他·旧事重提篇
2022年的第一节晚自习下课，四班像往日一样嘈杂，随着上课铃的打响，四班逐渐安静下来，这十分不对劲，并开始发出声音寻问周围同学，“鸡哥上哪去了？”“你看见没？”没错，此人正是鸡哥的同桌阿牛。从第二节晚自习开始，阿牛就没见过鸡哥，并且问了一圈之后，没有一个人知道他到底去哪了。学校听闻了这件事，急忙叫出砖家。“想在一定区域内让人凭空消失是不可能的，除非他升华了。”门卫大爷说到“全班49人除他一个剩下全部不知道他的踪影，这也666了。”先王说道，此时班级有四种说法：

1. 鸡哥死厕所了
2. 鸡哥被食堂抓走变成鸡腿饭了
3. 鸡哥去篮球场玩了
4. 鸡哥去问题了

第三节自习课，鸡哥仍然没有回来，连根鸡毛都没看见，先王听说这件事，从办公室来到班级介入调查。她派了 `小天气` `小峰瑞` `曹小豪` `屁凯` 等几位去往厕所，化学办公室，物理办公室，英语办公室等地。专家们在学校内秘密搜查。十分钟后，课代表们都已回来，并带来同一个消息：“没有，老师说他没来过。”专家们也没有找到任何线索，直到否定 `1 2 4` 三种说法，只剩下第 `2` 种说法，砖家们抱着不可能的思想去尝试，不出所料，鸡哥没有在食堂后厨，对此，厨师表示：“你们纯闲出屁来了，都晚上八点了，鬼才来吃饭，还说我杀害什么鸡哥，我连个学生影都没见到，管你什么鸡哥鸭哥，滚。”

最后经调查发现：说法 `4` 对一半，鸡哥去别的老师办公室问题去了。

> 花絮：鸡哥回来后，被先王重打 `20` 大板，对此鸡哥表示：我走行不？
